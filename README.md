# Student Info Formatter
This program:
- Collects a user's first name, surname, age, and favourite number.
- Displays a welcome message using f-strings.
- Converts the user's name to uppercase and title case.
- Calculates the user's age in months.
- Rounds the favourite number to two decimal places.
- Displays the data type of each input.
